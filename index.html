constructor

src/classes/Gaussian2D.js
constructor(options: object)
Parameters
options (object = {})
Name Description
options.height number (default 4*LN2/(PI*xFWHM*yFWHM)) Define the height of the peak, by default area=1 (normalized).
options.fwhm number (default 500) Full Width at Half Maximum in the number of points in FWHM used if x or y has not the fwhm property.
options.x object? Options for x axis.
options.x.fwhm number (default fwhm) Full Width at Half Maximum in the number of points in FWHM for x axis.
options.x.sd number? Standard deviation for x axis, if it's defined options.x.fwhm will be ignored and the value will be computed sd Math.sqrt(8 Math.LN2);
options.y object? Options for y axis.
options.y.fwhm number (default fwhm) Full Width at Half Maximum in the number of points in FWHM for y axis.
options.y.sd number? Standard deviation for y axis, if it's defined options.y.fwhm will be ignored and the value will be computed sd Math.sqrt(8 Math.LN2);
