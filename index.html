src/fwhmToInflectionPointsWidth.js

Compute the value of width between the inflection points of a specific shape from Full Width at Half Maximum (FWHM).

fwhmToInflectionPointsWidth(fwhm: number, kind: (number | string), options: any)
Parameters
fwhm (number) Full Width at Half Maximum.
kind ((number | string) = 1) kind may be 'gaussian', 'lorentzian' or 'pseudovoigt' or their corresponding number
options (any = {}) options
Name Description
options.mu number (default 0.5) ratio of gaussian contribution. It is used if the kind = 3 or pseudovoigt.

gaussian

src/gaussian.js

Calculate a normalized gaussian shape

gaussian(options: object): object
Parameters
options (object = {})
Name Description
options.height Number (default 1) maximum value of the curve.
options.normalized Number (default false) If it's true the area under the curve will be equal to one, ignoring height option.
options.fwhm number (default 500) number of points in Full Width at Half Maximum, Standard deviation will be computed as fwhm / 2 / sqrt(2 ln(2))
options.sd number? Standard deviation, if it's defined fwhm parameter will be ignored.
options.factor number (default 4) Number of time to take fwhm to calculate length. Default covers 99.99 % of area.
options.length number (default fwhm*factor+1) total number of points to calculate
Returns
object: {fwhm, data } - An object with the number of points at half maximum and the array of y values covering the 99.99 % of the area.
