gaussian

src/gaussian.js

Calculate a normalized gaussian shape

gaussian(options: object): object
Parameters
options (object = {})
Name Description
options.height Number (default 1) maximum value of the curve.
options.normalized Number (default false) If it's true the area under the curve will be equal to one, ignoring height option.
options.fwhm number (default 500) number of points in Full Width at Half Maximum, Standard deviation will be computed as fwhm / 2 / sqrt(2 ln(2))
options.sd number? Standard deviation, if it's defined fwhm parameter will be ignored.
options.factor number (default 4) Number of time to take fwhm to calculate length. Default covers 99.99 % of area.
options.length number (default fwhm*factor+1) total number of points to calculate
Returns
object: {fwhm, data }

getShape

src/getShape.js

Generate a shape of the specified kind

getShape(kind: (number | string), options: object): object
Parameters
kind ((number | string) = 1) kind may be 'gaussian', 'lorentzian' or 'pseudovoigt' or their corresponding number
options (object = {})
Name Description
options.fwhm number (default 500) number of points in Full Width at Half Maximum, Standard deviation will be computed as fwhm / 2 / sqrt(2 ln(2))
options.factor number (default 3) factor of standard deviation to increase the window size, the vector size is 2 factor sd
options.length number (default fwhm*factor+1) total number of points to calculate
Returns
object: {fwhm, data }

pseudoVoigt

src/pseudoVoigt.js

Calculate a linear combination of gaussian and lorentzian function width an same full width at half maximum

pseudoVoigt(options: object): object
Parameters
options (object = {})
Name Description
options.height Number (default 1) maximum value of the curve.
options.normalized Number (default false) If it's true the area under the curve will be equal to one, ignoring height option.
options.fwhm number (default 1000) number of points in Full Width at Half Maximum, Standard deviation for gaussian contribution will be computed as fwhm / 2 / sqrt(2 ln(2))
options.mu number (default 0.5) ratio of gaussian contribution.
options.factor number (default Math.tan(Math.PI*(0.9999-0.5))) Number of time to take fwhm to calculate length. Default covers 99.99 % of area.
options.length number (default fwhm*factor+1) total number of points to calculate
Returns
object: {fwhm, data }
